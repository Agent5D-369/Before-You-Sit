# Deployment

1. Upload the full folder to your GitHub repository root.
2. In GitHub Pages, publish from the main branch root.
3. Confirm your custom domain if you use one.
4. Test all Formspree flows on the live domain.
5. Replace placeholder witness quotes and placeholder art as approvals arrive.

## Important
- The result pages rely on query strings. Test copy/share behavior on mobile and desktop.
- If you change the domain, update `robots.txt`, `sitemap.xml`, and any canonical metadata.
